# Contribuições

Obrigado por considerar contribuir para o projeto!

## Como Contribuir

1. **Faça um Fork** do repositório.
2. **Crie uma branch** para sua feature ou correção (`git checkout -b minha-feature`).
3. **Faça as alterações** necessárias e **commit** (`git commit -am 'Adiciona uma nova feature'`).
4. **Push** para a branch (`git push origin minha-feature`).
5. **Crie um Pull Request**.

## Relatar Problemas

Se você encontrar algum problema, por favor, abra uma **Issue** no repositório com uma descrição detalhada do problema e, se possível, uma reprodução passo a passo.

Obrigado por ajudar a melhorar o projeto!
